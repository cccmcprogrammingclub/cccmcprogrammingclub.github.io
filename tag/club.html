Welcome to CCCMC's programming club. Strap in, because we will be moving fast.

Goals

  • Learn the basics of computer programming languages and to learn by doing
  • Provide time and atmosphere to collaborate and create your own projects
  • Provide access to relevent infromation for computer programming classes and careers
  • Make things!

We will use the Python 2.7 programming language. Python 3 is out and works great, but for the purposes of learning we will use the older code since it has more robust support online.

Note that I am only requiring Python to be used for learning purposes. Once you are on your own projects, I do not care which language you use. Ask me if you need guidance in choosing a language.

Learning

The following links are useful for taking online classes, tutorials, or for code reference. I suggest taking a class in Codecademy or Udacity in your spare time as long as it does not affect your school work! School comes first!

  • Codecademy
  • Udacity
  • Command Line Class This is a class to teach you the basics of the Bash command line, which is included in most linux/unix and Mac (OSX) distributions. It is a must know for working in programming, systems, and networking. Google, for example, wants its employees to know linux command line. This will feel like you have been transported back into the 1970s.

  • Windows Command Line Class Like the above, but for windows command line.

  • CCCMC Programming Club Github I will put code snippets here.

Text Editors

  • Sublime Text 3 - I suggest this text editor, as I will often use it during class.
  • Notepad++ - If you hate sublime for some reason, Notepad++ is also great
  • JetBrains If you want to bug JetBrains for a free educational copy of their IDE (Interactive Development Environment), feel free. I will not be using these on the projector due to their complexity.

Tools

  • Git Git is a version control system. I'll write a tutorial sometime, but there are tons of them online. Once you get comfortable coding, start using Git to store your progress. It makes life infinitely simpler.
  • Github This is a place to host and store your code in a git repository. Make a github account so I can see your code and help out! :)

Gamedev

I know many of you want to make games. I suggest starting with Pygame for the basics, or Unity if you want a more full featured program. I do not know Unity, so I cannot really help you there.

  • Pygame
  • Unity

Note that programming can quickly get overwhelming. This is a relatively new field in the long run of things, and it advances quickly. Try not to be too intimidated, but you will learn something. Go at your own pace and figure this stuff out and see if you like it. I will be here to assist.
